# Website Overlay

**Click any UI element. Describe what should change. Hand it to your AI coding tool in one paste.**

A browser extension that lets you point at things on any website — your localhost, staging, or live production — annotate them, and ship the annotations to Claude Code, Cursor, Windsurf, or whatever AI tool you use.
